1. USB メモリからパソコンを起動する

Linux Mint をインストールするには、まず USB メモリからパソコンを起動する必要があります。
ふだんパソコンの電源を入れると Windows が起動しますが、ここでは Windows の代わりに USB メモリから起動させます。

方法は大きく2つあります。お使いのパソコンに合った方法をお試しください。

方法 A: 起動メニューを使う（かんたん・おすすめ）

パソコンの電源を入れた直後に特定のキーを押すと、「どこから起動するか」を選ぶメニューが表示されます。ここで USB メモリを選ぶだけです。

手順:

  1. USB メモリをパソコンに差し込みます
  2. パソコンの電源を入れます（すでに起動している場合は「再起動」します）
  3. 画面に何か表示されたら、すぐに下の表にあるキーを 連打 してください（1回ではなく、トントントンと何回も押します）
  4. 起動メニューが表示されたら、USB と書かれた項目を矢印キー（↑↓）で選び、Enter キーを押します
🐧 ポイント
USB の項目は「USB HDD」「USB Flash Drive」「UEFI: KIOXIA」「USB Storage Device」など、メーカーによって表示が異なります。「USB」という文字が含まれているものを選んでください。

メーカー別 起動メニューキー一覧:

Dell（デル）

起動メニューキー: F12

  1. パソコンの電源を入れたら、Dell のロゴが出る前から F12 を連打します
  2. 「Boot Menu」または「起動メニュー」と書かれた画面が表示されます
  3. 一覧から「USB Storage Device」または「UEFI: KIOXIA〜」と書かれた項目を矢印キーで選びます
  4. Enter を押すと USB メモリから起動します

※ OptiPlex、Inspiron、Latitude、Vostro すべて共通です

  2. BIOS 設定画面が開いたら、矢印キーで「起動」または「Boot」タブに移動します
  3. 「USB」の項目を一番上に移動します（F6 で上に移動できる機種が多いです）

⚡ 起動メニューが出ないとき
キーを押すタイミングが遅いと Windows が通常起動してしまいます。
電源を入れ直して、もう少し早いタイミングで連打してみてください。
ロゴが表示される前から押し始めるのがコツです。

方法 B: BIOS 設定で起動順序を変更する

方法 A の起動メニューがうまく表示されない場合は、BIOS（バイオス）の設定画面から起動順序を変更します。

BIOS 設定画面を開くキー:

メーカー別 BIOS 設定キー

Dell: F2 ／ HP: F10 ／ Lenovo: F1 または F2 ／ 富士通: F2 ／ NEC: F2 ／ 東芝/dynabook: F2 ／ ASUS: Del または F2 ／ Acer: F2 ／ Panasonic: F2

BIOS 設定画面での操作手順:

  1. USB メモリをパソコンに差し込みます
  2. パソコンの電源を入れたら、上の表にあるキーを連打します
  3. BIOS 設定画面が表示されます（青い画面や灰色の画面が多いです）
  4. 矢印キー（←→）で 「Boot」「起動」 と書かれたタブに移動します
  5. 「Boot Priority」「起動順序」 という項目を見つけます
  6. 「USB」と書かれた項目を一番上に移動します
    （移動方法は画面の下や右側に説明が書いてあります。F6/F5 や +/- で上下する機種が多いです）
  7. F10 を押すと「設定を保存して終了しますか？」と聞かれるので、「Yes」を選んで Enter を押します
  8. パソコンが再起動し、USB メモリから Linux Mint が起動します
🐧 インストール後は元に戻してください
Linux Mint のインストールが完了したら、同じ手順で BIOS 設定を開き、起動順序を元に戻しておくと安心です（「HDD」や「SSD」を一番上に）。戻さなくても USB メモリを抜いておけば通常通り起動します。
⚡ 「Secure Boot」が有効になっている場合
一部のパソコンでは「Secure Boot」（セキュアブート）が有効になっていると、USB メモリから起動できないことがあります。
BIOS 設定画面で「Security」や「Boot」タブの中にある 「Secure Boot」を「Disabled」（無効）に変更してから、もう一度お試しください。

お使いの機種のキーがわからないとき

上の一覧にないメーカーや、キーを押しても反応しない場合は、次の方法をお試しください。

  1. YouTube で「（お使いの機種名） BIOS USB ブート」と検索すると、動画で手順を見ることができます
  2. ChatGPT などの AI に「（お使いの機種名）で USB メモリから起動する方法を教えてください」と聞くと、具体的な手順を教えてくれます
  3. パソコンの電源を入れた直後に画面の隅に「Press F12 for Boot Menu」などと一瞬表示されることがあります。その表示されたキーを押してください

2. Linux Mint Xfce をインストールする

USB メモリからパソコンが起動すると、Linux Mint のデスクトップ画面が表示されます。
この時点ではまだインストールされていません。まず試しに使ってみることもできます。

1

「Install Linux Mint」をダブルクリック

デスクトップ上にある 「Install Linux Mint」 というアイコンをダブルクリックします。
インストーラーが起動します。

2

言語の選択

左の一覧から 「日本語」 を選んで「続ける」をクリックします。

3

キーボードの選択

「日本語」 が選ばれていることを確認して「続ける」をクリックします。
英語キーボードの場合は「English (US)」を選びます。

4

マルチメディアコーデック

「マルチメディアコーデックをインストールする」に チェックを入れて「続ける」をクリックします。
これを入れると、動画や音楽ファイルを再生できるようになります。

5

インストールの種類

「ディスクを削除して Linux Mint をインストール」 を選んで「インストール」をクリックします。

⚡ 重要
この操作により、パソコン内の Windows やすべてのデータが削除されます。
必要なデータは事前にバックアップしてください。
6

タイムゾーンの選択

地図上で 「Tokyo」 が選ばれていることを確認して「続ける」をクリックします。

7

ユーザー情報の入力

以下を入力して「続ける」をクリックします。

  • あなたの名前: お好きな名前（日本語でも OK）
  • コンピューターの名前: 自動で入力されます（そのままで OK）
  • ユーザー名: 半角英数字で入力します（例: taro）
  • パスワード: 忘れないものを設定してください（このパスワードは後で使います）
8

インストール完了

インストールが始まります（10〜30分程度かかります）。
「インストールが完了しました」と表示されたら 「今すぐ再起動する」 をクリックします。
画面に「USB メモリを抜いて Enter を押してください」と表示されたら、USB メモリを抜いて Enter を押します。

🐧 おめでとうございます！
再起動すると Linux Mint Xfce のログイン画面が表示されます。
先ほど設定したパスワードでログインしてください。
次は Wi-Fi の接続と zram チューニングに進みましょう。

3. zram チューニングで高速化する

メモリ 4GB のパソコンでも体感 6〜7GB 相当で快適に動作するようにチューニングします。
ガイドカードの QR コードからこのページを開いている方は、ここからスタートしてください。

1

Wi-Fi に接続する

画面右下のネットワークアイコン（扇形のマーク）をクリックして、Wi-Fi に接続してください。
有線 LAN ケーブルで接続している場合はこのステップは不要です。

2

ターミナルを開く

画面左下のメニュー → 「システム」 → 「ターミナル」（黒い画面のアイコン）をクリックします。
または、キーボードで Ctrl + Alt + T を同時に押すと開きます。

3

セットアップコマンドを実行する

以下のコマンドをコピーして、ターミナルに貼り付けて Enter を押してください。

curl -sL https://penguin-computer.github.io/setup.sh | sudo bash

パスワードを聞かれたら、インストール時に設定したパスワードを入力します。
（入力中は画面に何も表示されませんが、入力されています。そのまま Enter を押してください）

4

完了！

「セットアップ完了」と表示されたら終了です。再起動後も設定は維持されます。

設定内容:

  • zram（メモリ圧縮）の有効化
  • 4GB スワップファイルの作成
  • swappiness の最適化（値: 15）
🐧 このチューニングの効果
メモリ 4GB のパソコンが体感 6〜7GB 相当に。LibreOffice の起動が最大 64% 高速化。
追加のハードウェア購入は不要です。
🐧 スクリプトについて
このスクリプトは Penguin Computer がオープンに公開しています。
Linux Mint のほか、Ubuntu、Xubuntu、Lubuntu、その他 Debian 系 Linux でもご利用いただけます。
USB メディアを購入されていない方もお気軽にどうぞ。

4. 日本語入力を設定する

Linux Mint Xfce をインストールした直後は、日本語の表示はできますが、日本語の入力（ローマ字変換）ができない場合があります。
以下の手順で日本語入力を設定しましょう。

1

日本語入力ソフトをインストールする

ターミナルを開いて（Ctrl + Alt + T）、以下のコマンドを実行します。

sudo apt install -y fcitx-mozc fonts-ipafont

これにより以下がインストールされます:

  • fcitx-mozc: Google 日本語入力と同じ変換エンジンを使った日本語入力ソフト
  • fonts-ipafont: きれいな日本語フォント（IPA フォント）
2

入力方式を設定する

続けて以下のコマンドを実行します。

im-config

設定画面が表示されます。「OK」→「Yes」→ 一覧から 「fcitx」 を選んで「OK」→「OK」の順に進めてください。

3

再起動する

設定を反映するために、パソコンを再起動してください。

sudo reboot
🐧 日本語入力の切り替え
再起動後、半角/全角 キーまたは Ctrl + Space で日本語入力のオン・オフを切り替えられます。

5. システムをアップデートする

インストール直後は、セキュリティパッチやソフトウェアの更新が溜まっていることがあります。
以下のコマンドで最新の状態にアップデートしましょう。

1

アップデートコマンドを実行する

ターミナルを開いて、以下のコマンドを実行します。
インターネット接続と処理に数分〜十数分かかります。

sudo apt update && sudo apt upgrade -y && sudo apt dist-upgrade -y && sudo apt autoremove -y && sudo apt autoclean -y

途中で確認画面が出た場合は、そのまま Enter を押してください。

🐧 定期的にアップデートしましょう
セキュリティを保つため、月に1〜2回はこのコマンドを実行することをおすすめします。
画面右下のシールドアイコンに通知が表示されることもあります。

6. ファイアウォールを有効にする

Linux Mint にはファイアウォール（UFW）が最初から入っていますが、初期状態では無効になっています。
有効にすることで、外部からの不要なアクセスをブロックできます。

1

ファイアウォールを有効にする

ターミナルで以下のコマンドを実行します。

sudo ufw enable

「Firewall is active and enabled on system startup」と表示されれば完了です。

2

状態を確認する（任意）

正しく有効になっているか確認したい場合は、以下のコマンドを実行します。

sudo ufw status

「Status: active」と表示されていれば OK です。

🐧 ファイアウォールについて
初期設定で外部からのアクセスはすべてブロックされます。
Web 閲覧やメールなど、ふだんの使い方に影響はありません。
ウイルス対策ソフトは不要です（Linux はその仕組み上、Windows のようなウイルス感染はほとんど起きません）。

7. バッテリー状態を確認する（ノートPC）

ノートパソコンをお使いの場合、バッテリーの劣化具合を確認できます。

1

バッテリー情報を表示する

ターミナルで以下のコマンドを実行します。

upower -i /org/freedesktop/UPower/devices/battery_BAT0

表示される項目のうち、特に重要なのは以下の3つです:

  • energy-full: 現在の満充電容量（Wh）
  • energy-full-design: 設計時の満充電容量（Wh）
  • capacity: バッテリー健康度（%）— 80% 以上なら良好です
🐧 バッテリーの目安
capacity が 80% 以上であれば十分実用的です。
50% を下回っている場合は、電源アダプターを常時接続してのご利用をおすすめします。

8. 困ったときは

USB メモリから起動できない

  • 起動メニューキーを押すタイミングが遅い → 電源を入れ直して、もっと早く連打してください
  • USB メモリが認識されない → 別の USB ポートに差し替えてみてください（USB 2.0 のポートがある場合はそちらを試してください）
  • Secure Boot が有効 → BIOS 設定で「Secure Boot」を「Disabled」に変更してください
  • UEFI / Legacy の設定 → BIOS 設定で「UEFI」モードになっていることを確認してください。「Legacy」や「CSM」モードでは起動できない場合があります

Wi-Fi が見つからない

  • 一部のノートパソコンでは、Wi-Fi が自動で認識されないことがあります
  • その場合は有線 LAN ケーブルでインターネットに接続してから、システムアップデートを実行すると Wi-Fi が使えるようになることがあります

画面の表示がおかしい

  • 画面が大きすぎる・小さすぎる場合: メニュー → 設定 → ディスプレイ で解像度を変更できます
  • 画面がちらつく場合: 「ドライバーマネージャー」から推奨ドライバーをインストールしてください
